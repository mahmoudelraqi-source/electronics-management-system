# نظام إدارة المحترف للإلكترونيات

نظام متكامل لإدارة المخازن والحسابات والمبيعات لمركز المحترف للإلكترونيات.

## ✨ الميزات

- 🔐 **نظام الأمان**: تسجيل دخول آمن مع تشفير كلمات المرور
- 📦 **إدارة المخازن**: إضافة وتعديل المنتجات مع تنبيهات المخزون المنخفض
- 🛒 **نظام المبيعات**: إنشاء فواتير مع تحديث تلقائي للمخزن
- 💰 **الحسابات**: تتبع الديون والمبيعات الآجلة
- 📁 **الأرشفة**: أرشفة شهرية للمبيعات والفواتير
- 📱 **واجهة متجاوبة**: تعمل على الموبايل والكمبيوتر
- 🌍 **واجهة عربية**: دعم كامل للغة العربية (RTL)

## 🚀 البدء السريع

### المتطلبات
- Bun (أحدث إصدار)
- PostgreSQL

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/mahmoudelraqi-source/electronics-management-system.git
cd electronics-management-system

# تثبيت المكتبات
bun install

# تشغيل قاعدة البيانات
bun run src/db-init.ts

# تشغيل التطبيق
bun run src/api.ts
